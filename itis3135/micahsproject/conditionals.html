Operators & Conditionals

When comparing variables mathematically, Java supports the following operators.

Operator Syntax Result if a = 1 & b = 2 Result if a = 1 & b = 1
Greater than a > b false false
Less than a < b true false
Greater than or equal to a >= b false true
Less than or equal to a <= b true false
Equal to a == b false true
Not equal to a != b true false

Up until now, boolean values haven’t really seemed to have any important use. However, when combined with the power of conditionals, that changes.

If Statements

In Java, if statements can be used to decide whether or not to execute a portion of code. The syntax for if statements is as follows:

if(conditonal)

{

  (code to be executed)

}

Let’s take a very simple example. Say you want to divide two int variables, “a” and “b”. However, if the second of the int variables is a 0, division will be impossible. We can use if statements to check to see if that variable is, in fact, 0.

if (b > 0)

{

  System.out.println(“Result: ” + a / b);

}

If “a” was assigned a value of 8, and “b” was assigned a value of 4, the output would be:

Result: 4

However, if “b” was 0, no matter what “a” was, nothing would be printed. This might be a bit confusing for anyone using our program to try and divide numbers. We can clear up that confusion by using an else statement.

Else statements can be attached to the end of if statements, and will run if the corresponding if statement returns false. In our example, we could print an error message to the console with an else statement like this:

if (b > 0)

{

  System.out.println(“Result: ” + a / b);

}

else

{

  System.out.println(“Error! You can’t divide by zero!”);

}

Let’s take another example. Say we need to figure out whether an integer “a” is between 1 and 10. We could put an if statement inside another, with the first one checking to see if the integer is greater than 1, and the second checking to see if the integer is less than 10 (or vice versa). This solution would look something like this:

if (a > 1)

{

  if (a < 10)

  {

    System.out.println("a is between 1 and 10");

  }

}

else

{

  System.out.println("a is not between 1 and 10");

}

There is, however, a more efficient method to do this. We can optimize this code by using logical operators. Logical operators can be used to extract a single boolean value (which if you remember, means either true or false) from multiple comparison statements.

Operator Syntax Result if a = false & b = false Result if a = true & b = false Result if a = true & b = true
AND a && b false false true
OR a || b false true true
NOT !(a && b) true true false

We can combine these logical operators with the comparison operators to make this code more efficient:

if (a > 1 && a < 10)

{

  System.out.println("a is between 1 and 10");

}

else

{

  System.out.println("a is not between 1 and 10");

}
