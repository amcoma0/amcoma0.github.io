Variables

In all programming languages, “variables” are containers that are used to store data for a program. A variable could be used to store how many of a particular item is in stock on an online store. Variables can also be used to store the name of that item, or how much that item costs. There are five main types of variables in Java:

Variable Type What is it? Examples
int The int variable type is used to store integers. An int can be any whole number, whether positive or negative. 4 -73 56843 -800 0
float The float variable type stores floating point numbers (aka decimal numbers). 3.1415 7.00 364.892 2.5 0.1
char The char type stores a single character. It can be a letter, a number, or any other symbol. a G 6 % $
string The String variable types hold text (or a “string” of characters). It can be used to store words or sentences “Hello World!” “How are you?” “Items left: 75” “John Doe” “Green”
boolean The boolean variable type can only hold two values: true or false. true false

The most common way to make, or instantiate, a variable in Java is as follows:

variableType variableName = value;

In the above code, “variableType” should be the type of variable we want (int, String, etc.). Next, “variableName” is whatever we want the name of the variable to be. This is what we’ll use in the future to refer to whatever data is stored in that variable. Finally, “value” is what is actually held by the variable; the data itself.

Here are a few examples of declaring actual variables in Java:

String greeting = “Hello there!”;

int livesLeft = 5;

boolean isGameOver = true;

Note! The end of a line of code in Java is virtually always indicated by a semicolon (;), don’t forget that!

You can even use variables to declare other variables.

int myApples = 3;

int yourApples = 7;

int totalApples = myApples + yourApples;

In this case, “totalApples” would be assigned a value of 10.

Printing

Throughout the material provided on this site, information within segments of code will be output using the “System.out.println() method. The specifics of why this method functions the way it does are a bit outside the scope of what this site teaches, but it works as follows:

System.out.println(“Hello World!”);

With the above line of code, the message “Hello World!” will be printed to the system’s console. This method can be used with variables as well.

int totalApples = 10;

System.out.println(“There are ” + totalApples + “ apples in all.”);

This code will print the following to the console:

There are 10 apples in all.

The plus signs on either side of the “totalApples” variable are used to string together (or concatenate) the value stored in the variable with the text on either side. So the console prints “There are “, then the value of “totalApples” (10), finally followed by “ apples in all.”
