Arrays

There is another data type that was not covered in the Variables page, and that is an array. Arrays are used to essentially hold a list of other data types. For example, let’s say you need to store the first names of 10 employees. What you might think of doing is declaring 10 separate String variables and assigning one of the names to each of them. However, arrays make that much simpler. You can declare arrays as follows:

variableType[X] arrayName;

In this statement, “X” would be replaced with the length of the array. An array’s length determines how many items it can store inside of it. So, to create an array called “employeeNames” to store 10 names, we would use the above statement followed by a comma separated list of each item we want to put inside the array, surrounded by curly braces:

String[10] employeeNames = {"Dwight", "Jim", "Pam", "Michael", "Kevin", "Andy", "Phyllis", "Stanley", "Oscar", "Angela"};

In order to actually access the values within an array, we use the array name immediately followed by the index of our desired value in square brackets. Now, one very important thing to remember about arrays in programming is that the index of an array does NOT start at 1, but at 0. What does this mean? Well, in the code below, try and take a guess as to what value the String variable “myName” will be assigned.

String myName = employeeNames[1];
